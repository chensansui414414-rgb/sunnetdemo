# 朝霞 / 晚霞 / 火烧云智能预测 PRD

- 版本：1.0
- 用户：摄影爱好者、追云者、零基础用户
- 目标：用简洁日历界面展示未来三天霞光潜力，并以光路与云层几何关系解释预测。
- 数据优先级：GFS 全球基础场；HRRR 北美 3km 局地修正；CAMS AOD。
- 核心算法：日落方向低云、能见度和 AOD 构成光照通道；本地 30%—70% 中高云、云底高度与光学厚度构成画布；两者采用物理门控相乘。
- 降级策略：GRIB、网络或重型依赖失败时返回确定性 Mock 数据，页面和 API 不崩溃。
- 视觉：极简黑白灰、克制霞光橙、宽松留白、紧字距、16—20px 圆角、CSS 优先轻动效。
- 成功标准：桌面与手机均可阅读；三日预测随日期和坐标变化；卡片可展开；反馈可保存；滚动揭示与数字动画生效。
